# Track Diagrams

Track Diagrams is an interactive web application for visualizing and managing railway track diagrams. It enables users to explore route layouts, inspect track sections, and customize the view with various settings and overlays. **Not to be used for any safety-critical processes or official purposes.**

## Features

- **Route Selection**: Choose different available routes and view their layouts.
- **Track Visualization**: Graphical representation of tracks, including attributes such as electrification status, connections, and track identifiers (TID). Tracks on alternative routes are visually distinguished.
- **Settings Panel**:
  - **View Controls**: Adjust window size (miles), yards per pixel, grid spacing, and toggle overlays and alternate ELR rulers.
  - **Diagram Management**: Edit diagrams using the provided interface.
- **Controls**:
  - Center view on a given ELR (Engineerâ€™s Line Reference), mile, or yard.
- **Tracks Table**:
  - View and filter a detailed tabular list of tracks for the selected route.
  - Inspect track extents, electrification, and section data.

## Usage

1. Open `index.html` in a modern web browser.
2. Select a route using the sidebar.
3. Adjust settings for your viewing preference.
4. Interact with controls to center on a specific section or ELR.
5. View or edit diagram data as needed.

## Development

- Written in **JavaScript**, with main code in `script.js`, UI handling in `ui.js`, and styles in `style.css`.
- Route and track data structure can be found in `data.js`.
- Utilities for development (e.g., task automation) found in `.vscode/tasks.json`.
